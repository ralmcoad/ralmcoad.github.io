갑상선 항진증

이론

갑상선중독증 ⊃갑상선기능항진증 ⊃그레이브즈병

  • 중독증: 혈액에 갑상선호르몬이 많은 상태
  • 항진증: 갑상선호르몬 과생산(방사성 요오드 섭취율이 증가됨)
  • 그레이브즈: TSHR Ab. 때문에 갑상선호르몬이 과잉생성되는 질환.

#1. 그레이브즈 vs 독성 결절 갑상선종(Toxic nodular goiter)

  1. 그레이브즈: TSHR Ab가 갑상선 전체를 자극→ diffuse goiter/RAI uptake
  2. 독성[1] 결절: 갑상선 호르몬을 과생산하는 결절→ nodular goiter/RAI uptake
  • 치료: 메티마졸 or 프로필티오유라실(임신 첫 3개월 이내) → 불응시 방사성요오드/절제술

#2. 갑상샘염

  • 자가면역/감염/의인성 등으로 갑상샘에 염증이 생기는 질환을 통틈.
  • 경과: 중독증(세포파괴로 호르몬이 혈액으로 흘러나옴) → 저하증(세포기능저하) → 회복(대부분 자연회복됨)
  1. 아급성 갑상샘염[2]: 압통성 부은 갑상선+ ↑ESR ± 최근 감기 → NSAID/베타차단제
  2. 무통성/산후 갑상샘염[3]: 하시모토 갑상샘염의 이형(↑anti-TPO Ab) → 베타차단제
  • 치료: 중독증에 대한 대증치료로 베타차단제를 쓰며, 항갑상선제(e.g. 메티마졸)는 금기.

#3. 의인성

  1. 레보티록신 과다: ↓Thyroglobulin(Tg)
  2. ADR: 아미오다론 & 리튬
  • 치료: 약물중단

#4. 합병증

  1. 저칼륨성 주기성마비(Thyrotoxic Periodic Paralysis): 음주/과식/운동 이후 항진증 증상과 함께 나타나는 갑작스런 근위약. 급성기치료는 딱히 없고(칼륨보충X) 자연호전됨.
  2. 갑상선 중독 위기/발작(Thyrotoxic Crisis/Storm): 갑상선중독증의 치명적 악화. 의식저하/발열/빈맥. 치료는 4P(Propranolol, PTU, Prednisolone, Potassium iodide)

대표유형

대표유형: 두근거림/빈맥(+) 체중감소(+) 안절부절(±) 더위(±) 고혈압(±)[4] 심전도사진(±)

  • 검사는? → 갑상샘기능검사(유리 티록신 and 갑상샘자극호르몬)

대표유형: ↑T4(+)[5] ↓갑상샘자극호르몬(+)

  • 진단은? → 갑상샘 항진증/중독증

    • if: 임신≤3개월(+)
      • 진단은? → 정상임신
      • 치료는? → 경과관찰

  • 검사는? → 스캔[6] (≈방사성요오드섭취율[7])

  • if: 갑상샘자극호르몬수용체항체(-) 결절성 목덩이(±) 일측성 사진(±)[8]

    • 검사는? → 스캔
    • 진단은? → 독성(=기능성) 결절
    • 치료는? → 방사성요오드 (RAI, 방사성동위원소치료)

  • if: 갑상샘자극호르몬수용체항체(+) 미만성 목덩이(±)[9] 눈돌출(±)[10] 양측성 사진(±)

    • 검사는? → 스캔[11]
    • 진단은? → 그레이브즈병
    • 치료는?
      • if: 임신≤3개월 → 프로필티오유라실
      • else: → 메티마졸[12] (≈카비마졸[13])
      • if: 약물치료 불응 → 방사성요오드/절제술

  • if: ↓티로글로불린(=갑상샘글로불린)

    • 검사는? → 스캔
    • 진단은? → 의인성 (레보티록신 과다투여)
    • 치료는? → 레보티록신 감량

  • if: 부은 압통성 갑상선(+) ↑적혈구침강속도(±) 최근 감기(±)

    • 검사는? → 스캔
    • 진단은? → 아급성(subacute)
    • 치료는?
      • 교감신경 증상에 대해서 → 베타차단제
      • 압통에 대해서 → NSAID/스테로이드[14]

  • if: 3~4개월전 출산(++) 갑상샘과산화효소항체(±) 갑상샘자극호르몬수용체항체(±)[15]

    • 검사는? → 스캔[16]
    • 진단은? → 산후/무통성 갑상샘염[17](=일과성갑상샘중독증을 동반한 만성갑상샘염)

  • if: 갑상샘과산화효소항체(+) 사진(±)

    • 검사는? → 스캔
    • 진단은? → 갑상샘염 (하시모토[18] likely, 무통성 possible, 아급성 unlikely)
    • 치료는? → 경과관찰/NSAID/베타차단제

대표유형1: 음주/과식/운동 이후 근력저하/마비(+) 최근 항진증 증상(+)

  • 진단은? → 갑상샘 주기 마비
  • 검사는? → 칼륨
  • 치료(예방)는? → 베타차단제

대표유형2: 음주/과식/운동 이후 근력저하/마비(+) K≤3.5(+) "목이 부어보인다"

  • 진단은? → 갑상샘 주기 마비
  • 검사는? → 갑상샘기능검사

대표유형3: 음주/과식/운동 이후 근력저하/마비(+)
↑T4(+) ↓갑상샘자극호르몬(+) 갑상샘자극호르몬수용체항체(+)

  • 진단은? → 갑상샘 주기 마비를 주소로 찾아온 그레이브즈 환자
  • 치료는? → 메티마졸

  1. (toxic vs non-toxic) = (functioning vs non-functioning) ≠ (malig. vs benign)↩︎

  2. 최신명칭 Subacute granulomatous thyroiditis↩︎

  3. 최신명칭 Subacute lymphocytic thyroiditis↩︎

  4. 그닥 헷갈리진 않지만, 두근거림 고혈압이랍시고 1' 알도스테론증이랑 헷갈리면 안된다. 참고로 갑상선으로 인한 고혈압은 140/80 같이 수축기-이완기 차이가 벌어지는게 특징이다.↩︎

  5. 가끔 T4가 high-normal로 출제된다. 굳이 따지자면 subclinical 갑상샘 항진증이지만, 국시적으로는 걍 똑같이 취급하고 치료하면 된다.↩︎

  6. 틀린 선지로 세침흡인/초음파/CT 등이 싹주어지며, 심지어 "갑상선자극호르몬수용체" 까지도 있었다. 즉 혈액검사보다 스캔이 우선되어야 한다는 얘기.↩︎

  7. 스캔이랑 비슷한 검사인데, 요즘은 스캔을 더 많이 한다고 한다.↩︎

  8. 사진 대신 "방사성요오드 섭취율" 수치를 주거나, 냉결절/열결절이라고 언급하기도 한다. 이때 갑상선염은 섭취율이 감소한다는 것을 알아야 풀수있는 문제가 기출되었다.↩︎

  9. 문제에서 "전체적으로 커져있다" 라고 자주 언급한다.↩︎

  10. 다른 항진증도 드물게 일으키지만, 거의 그레이브즈에 특이적.↩︎

  11. 문제들에서는 항체/티로글로불린/ESR 등을 먼저 보여줄 때가 많지만, 원칙적으로 무조건 스캔이 혈액검사보다 우선이다(TFT→스캔→TFT외 혈액검사). ↑T4 ↓TSH 상황에서 "다음에 해야할 검사는?" 은 100% 스캔이 정답이다.(그레이브즈에서 선지에 TSH 검사가 있어도 무조건 스캔이 이김) 이걸 강조하기 위해 일부러 if마다 "검사는?→스캔"을 각각 달아두었다.↩︎

  12. 애초에 메티마졸이 훨씬 더 좋은 약인데, 임신 첫 3개월만 기형성 때문에 PTU로 대체한다. 임신 3개월 이후는 다시 메티마졸 쓰면 됨↩︎

  13. 사실상 동일한 약물이다. 가끔 선지에 이걸로도 출제된다.↩︎

  14. 압통이 매우 심한 환자에서, 선지에 베타차단제, 스테로이드가 같이 있었는데, 스테로이드가 이겼다.↩︎

  15. 가끔씩 동반된다. 그레이브즈랑 혼동 주의↩︎

  16. 마찬가지로 틀린선지로 "갑상샘과산화효소항체" 가 있었다. 아무리 진단에 특이적이어도 스캔 먼저 하는게 우선이다.↩︎

  17. 병리학적 최신 명칭은 "Subacute lymphocytic thyroiditis"임에도 불구하고, 지속적으로 "아급성갑상샘염"은 틀린 선지로 제시되었었다. 애초에 우리가 말하는 "아급성갑상샘염"의 최신명칭은 "Subacute granulomatous thyroiditis"이기 때문에 모순은 없는거긴 하다.↩︎

  18. 혼동주의: 하시모토의 항진증 phase에는 보존적치료를, 저하증 phase에는 레보티록신을 줘야한다. 갑상선 저하증 참고.↩︎
