뇌하수체저하증

이론

(본 문서에선 2개 이상의 뇌하수체호르몬이 저하된 뇌하수체저하증만 서술한다. 이외의 뇌하수체저하증들은 ~뇌하수체 질환 모아보기 문서 참고)

  1. 부신겉질자극호르몬 결핍 → 뇌하수체성 부신저하
  2. 갑상샘자극호르몬 결핍 → 뇌하수체성 갑상선 저하증
  3. 난포자극/황체형성호르몬 결핍 → 성선저하증(hypogonadotropic hypogonadism)
  4. 인슐린유사성자인자-1 결핍 → 성장호르몬 결핍증
  5. 프로락틴(유즙분비호르몬) 결핍 → 수유부 말고는 딱히 불편함 없는듯.

대표유형

대표유형: ↓난/황/갑/부/인[1]/↑프<200(+) Mass effect(+)[2]

  • 진단은? → 뇌하수체저하증 w/ 고프로락틴혈증 d/t 비기능성 뇌하수체샘종[3]
  • 검사는? → 복합 뇌하수체 자극검사[4] → 뇌/안장MRI
  • 치료는? → 나비뼈경유절제술

대표유형: ↓난/황/갑/부/인/프(+)[5] Mass effect(-)[6]

  • 진단은? → (범)뇌하수체저하증[7]
  • 검사는? → 복합 뇌하수체 자극검사
  • 치료는?[8]
    • if: ↓부신겉질자극호르몬 → 글루코코티코이드(코티솔/히드로코티손) (가장 먼저 투여)[9]
    • if: ↓갑상샘자극호르몬 → 레보티록신
    • if: ↓난/황 → 에스트로겐(여), 테스토스테론(남)

대표유형: 산후출혈후 무월경(++) ↓난/황/갑/부/인/프(+)[5-1]

  • 진단은? → 쉬한증후군 (뇌하수체저하증)
  • 치료는?
    • if: ↓부신겉질자극호르몬 → 글루코코티코이드(코티솔/히드로코티손) (가장 먼저 투여)[10]
    • if: ↓갑상샘자극호르몬 → 레보티록신
    • if: ↓난/황 → 에스트로겐

대표유형: 냄새못맡음(++) 무월경/수염안남(+) ↓난/황(+)

  • 진단은? → 칼만증후군 (뇌하수체저하증)
  • 치료는?
    • ↓난/황 → 에스트로겐(여), 테스토스테론(남)

대표유형: 무월경(+) 다이어트/운동(±) ↓뼈나이(±) Low~Low normal 난/황/갑/부/인/프(+)[5-2]

  • 진단은? → 체질적 지연[11] (시상하부-뇌하수체기능이상)
  • 검사는? → 생자방 자극검사[12] → 뇌/안장 MRI
  • 치료는? → 경과관찰

  1. 모두 저하되어 있을 필요는 없고, 2개 이상이 동반저하 되어있으면 (+)이다. 예를들면 ↓난↓황=갑↓부=인 이런식.↩︎

  2. 두통/양귀쪽반맹↩︎

  3. Stalk effect: 커진 종양이 시상하부-뇌하수체 줄기를 눌러 도파민(프로락틴 억제 호르몬) 전달을 차단 → 프로락틴 분비가 완전히 억제되지 못해 경도-중등도 상승(<200). 선종 자체는 프로락틴을 분비하지 않음(‘비기능성’). 프로락틴>200 이면 프로락틴 분비성 종양을 의심하며, 고프로락틴혈증 문서 참고.↩︎

  4. 각종 뇌하수체호르몬들이 정말로 적은지 확진검사↩︎

  5. 마찬가지로 2개 이상 동반저하시 (+)↩︎↩︎↩︎

  6. 성인에서는 피로 등 비특이적 증상으로 내원해서, Lab을 잘봐야 한다. 소아청소년에서는 2차성징 늦음을 주소로 내원할때가 많다.↩︎

  7. 틀린선지로 "일차 갑상샘 저하증", "일차 부신저하증" 등이 같이 제시된다. 각각의 호르몬만 보면 그럴듯하지만 두개 이상이 저하되었다는 것이 포인트이다.↩︎

  8. 뇌하수체호르몬이 아니라, 말단장기의 호르몬을 투여함에 주의.↩︎

  9. 무월경 or 수염/음모없음 이 주소여서 에스트로겐/테스토스테론인척 낚지만, ↓부(+) 라면 무조건 코티솔 먼저 투여. 물론 부/갑이 정상이고 답으로 테스토스테론을 고르는 문제도 있었다.↩︎

  10. 무월경이 주소여서 에스트로겐인척 낚지만, ↓부(+) 라면 무조건 코티솔 먼저 투여.↩︎

  11. 체질/다이어트/과다한 운동 으로 인해 가역적으로 시상하부-뇌하수체 축이 기능저하 되는 질환↩︎

  12. 난/황이 정말로 적은지 확진검사↩︎
