무증상 단백뇨╱혈뇨

(다발골수종 vs 방광암도 드물게 무증상처럼 출제되지만, 따로 서술해 놓았다. 해당 문서 참고.)

정성적 기준[1]

  • 단백뇨: >150mg[2] (의심기준: 1+)
    • 사구체성 단백뇨: >1g (의심기준: 3+)
      • Nephrotic: >3.5g
      • Nephritic: <3.5g
  • 혈뇨: 적혈구≥3/고배율시야 (의심기준: 잠혈 1+)[3]
  • 농뇨: 백혈구≥5/고배율시야

대표유형: 무증상 단독 단백뇨(+)

  • 진단은? → 일과성/기립성 단백뇨
  • 검사는? → 정량검사/아침첫소변 정량검사
  • 치료는? → 경과관찰[4]

대표유형: 무증상 단독 혈뇨(+)[5]

  • 진단은? → 일과성 혈뇨
  • 검사는? → 재검[6]
  • if: 가족성 혈뇨/요석(+)
    • 진단은? → 특발성 고칼슘뇨증
    • 검사는? → 소변 칼슘/크레아티닌 비
    • 치료는? → 저염식[7] and/or 싸이아자이드
  • if: 신정맥압박 CT사진(+)
    • 진단은? → 호두까기(신정맥압박)

대표유형: 단백뇨(+) 혈뇨(+)

  • 검사는? → 단백뇨 정량검사[8] and 이형적혈구[9]
  • if: 정량검사 >1g or 이형적혈구 >75%
    • 진단은? → 사구체성 단백뇨/혈뇨
    • 검사는? → 신생검
  • else:
    • 진단은? → 일과성 단백뇨+혈뇨
    • 검사는? → 재검

  1. 실제 그 질환의 진단기준인 수치와, 이 정도 이상이면 의심해봐야한다 싶은 수치를 섞어 놓았다. 근데 제일 유용하게 섞어 놓음.↩︎

  2. 24시간 검사 및 단백질/크레아티닌 비 기준으로 150mg이며, 이를 알부민/크레아티닌 비로 환산하면 30mg이 기준이 된다. 가끔 알부민/크레아티닌 비 만 제시하는 경우 있으므로, 이것도 외워두자↩︎

  3. 문제에서 적혈구 0~5 이런식으로 줄 때도 있는데, 이건 혈뇨(+)로 취급하지 않는다.↩︎

  4. 저염식도 필요없음. 가끔 틀린선지로 제시됨.↩︎

  5. 소변검사의 첫 단계는 무조건 dipstick이고, 단백질 몇mg/적혈구 몇개 로 나오는 정량검사와 달리, 단백질(1+)/잠혈(1+) 이런식으로 검사결과가 나온다.↩︎

  6. "정기적 소변검사"라고도 주어진다. 참고로, "24시간 소변검사"가 흔히 틀린선지로 제시된다.↩︎

  7. Na를 덜 먹어야 Ca도 덜 빠져나간다고 한다.↩︎

  8. 24시간 단백뇨, 소변 단백질/크레아티닌 비, 소변 알부민/크레아티닌 비↩︎

  9. 사구체 손상으로 인해 발생하는 찌그러진 적혈구 → 사구체성 혈뇨 감별↩︎
