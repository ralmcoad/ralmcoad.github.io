MCD, PSGN, IgAN

교과서적으로는

  • Nephrotic synd.=단백뇨>3.5g
  • Nephritic synd.=이형적혈구>75% or 적혈구원주(+)
    로 나누지만, 99% 국시문제에서는 단백뇨도 정량값을 안주고(4+라고만 줌), 이형적혈구/적혈구원주도 안 준다. 남은 구분 방법은 혈뇨(-) 뿐이다. 혈뇨(+)라고 해서 신염이라고 말할 순 없지만, 혈뇨(-)면 무조건 신증이다.

대표유형: 단백뇨(+) 혈뇨(+)

  • 검사는? → 단백뇨 정량검사 and 이형적혈구 검사사
  • if: 정량검사 >1g or 이형적혈구 >75% (단, 최근 감기/C3 무언급)
    • 진단은? → 사구체성 단백뇨/혈뇨
    • 검사는? → 신생검
  • else:
    • 진단은? → 일과성 단백뇨+혈뇨
    • 검사는? → 재검

대표유형: 단백뇨(4+) 혈뇨(-)[1] ↓알부민<2.5g(+) ↑콜레스테롤 >250(+)

  • 진단은? → 신증후군
  • 보존적치료는?[2] → ACEi/ARB , 푸로세미드(고리이뇨제), 저염/저단백식
  • 병터는? → 발세포 발돌기
  • if: 소아(<12세)
    • 진단은? → 미세변화병 (신생검 없이 일단 진단)
    • 치료는? → 스테로이드
      • if: 스테로이드 불응 → 신생검(r/o 국소조각토리굳음증)
  • if: >12세
    • 진단은? → 미세변화병(less likely)[3]
    • 검사는? → 신생검
    • if: 전자현미경 발돌기 effacement
      • 진단은? → 미세변화병
      • 치료는? → 스테로이드
        • if: 스테로이드 불응 → 신생검(r/o 국소조각토리굳음증)
  • if: 혈역학적 불안정(+) 전신부종(+)
    • 진단은? → 저혈량성 위기
    • 치료는? → 20% 알부민
  • if: 스테로이드 치료중, 4대 염증수치(+) 반동압통(+) 복부팽만(+)
    • 진단은? → (자발성 세균성) 복막염[4]
    • 치료는? → 탁심트리악손 and 20% 알부민

대표유형: 단백뇨(+) 혈뇨(±)

  • 진단은? → 신증 or 신염 (문제에 따라 진단명 특정할수 있게 주기도 함)
  • if: 부종(+) 숨참(+) 폐부종/거품소리(±)
    • 치료는? → 푸로세미드(고리이뇨제)
  • if: 부종(-)
    • 치료는? → 경과관찰~저염/저단백식~ACEi/ARB
  • if: 단백질(4+) 혈뇨(-) ↓알부민<2.5g ↑콜레스테롤 >250
    • 치료는? → 스테로이드

대표유형: 일주일 전 감기(+)[5] 콜라색(검붉은색) 혈뇨(+)

  • 진단은? → 사슬알균신염(likely)
  • 검사는? → 항사슬알균용혈소O 검사 or C3 검사
  • if: ↓C3
    • 진단은? → 사슬알균신염
    • 치료는? → 경과관찰~저염/저단백식~ACEi/ARB[6]
  • if: C3 정상 or 단백질(4+)/>3.5g
    • 검사는? → 신생검

대표유형: 3일 전 감기(+) 선홍색~검붉은색 혈뇨(+) 단백뇨(±)[7]

  • 진단은? → IgA 신염(면역글로불린A 콩팥병)
  • 치료는? → 경과관찰~저염/저단백식~ACEi/ARB

대표유형: 선홍색~검붉은색 혈뇨(+) 단백뇨(±) 관절통(+) 자반(+)

  • 진단은? → 헤노흐-쉰라인 자반병[8]
  • 치료는? → 경과관찰[9] → NSAID → 스테로이드

  1. 잠혈(1+ ~2+) 정도가 가끔 동반될 수 있다. "12세 단백뇨(4+) 혈뇨(+) ↓알부민<2.5g ↑콜레스테롤 >250" 이면 생검을 할지 말지 애매한 상황이지만, 선지에 "미세변화병"/"스테로이드" 가 있으면 그걸 고르면 되고(약간 혈뇨가 동반될수 있음을 인정), 선지에 "신생검" 이 있으면 그걸 고르면 된다(높고 지속적인 혈뇨라면 미세변화병이 아닐거같으니 신생검을 해보자는 입장).↩︎

  2. 사구체가 손상됐으므로, 사구체 압력 감소 위해 ACEi/ARB를 쓰고, 부종 빼주기 위해 이뇨제, 신장 부담 줄이기 위해 저염/저단백식↩︎

  3. 소아보다는 유병율이 낮아 신생검으로 확진해줘야함, 그러나 답으로 미세변화병을 골라야하는 문제도 꽤 있다.↩︎

  4. 복수 d/t 신증후군에다가 면역저하 d/t 스테로이드가 더해져 복막염이 신증후군에서 종종 생긴다. 문헌에 따라 이것도 자발성세균성복막염으로 보기도 한다. 실제로 병리를 공유하며, 이는 간경변의 자발성 세균성 복막염의 치료도 탁심+알부민 이라는 사실과 일치한다.↩︎

  5. 사슬알균이 일으키는 다른 질병인 고름딱지증(impetigo) 등으로 문제가 나온적도 있다.↩︎

  6. 선지에 저염식과 경과관찰이 같이 있던적이 있는데, 답은 저염식이었다. 아주 증상이 없는거 아니면 보존적 치료를 해줘야한다는 마인드↩︎

  7. 보통 사구체질환(신증/신염)은 단백뇨(>1g)가 무조건 발생하는데, 오직 IgA신염만 단백뇨가 음성일 수도 있다. 만약 상황을 PSGN과 헷갈리게 줬는데 단백뇨(-) 였다면 무조건 IgA신염↩︎

  8. IgA 신염=콩팥에 IgA 면역복합체가 침착. HSP=전신 혈관에 IgA 면역복합체가 침착.(80%에서 콩팥도 침범하며, 콩팥 조직소견이 IgA 신염과 동일함.)↩︎

  9. "수분공급"이 답이었던적도 있다.↩︎
