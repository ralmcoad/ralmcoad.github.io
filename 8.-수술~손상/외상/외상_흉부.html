외상_흉부

(교통사고/낙상 등의 여러 외상 상황을 제시한다. 대표유형에는 제외하고 서술하였다.)

대표유형: 외상성 뇌출혈(+) 혈복강(+) 골절(+) 기흉(+)

  • 최우선 치료는? → 기흉 (Airway - Breathing - Circulation)

대표유형: 과감/호흡음감소(+) 피하기종(±) ↓산소포화도(±) 혈역학적 불안정(-)

  • 진단은? → 기흉 d/t 외상
  • 치료는? → 가슴관 삽입

대표유형: 과감/호흡음감소(+) 피하기종(±) ↓산소포화도(±) 혈역학적 불안정(+)

  • 진단은? → 긴장성 기흉
  • 치료는? → 바늘 감압[1]

대표유형: 가슴관삽입을 시행했으나 계속 숨이 차고 피하기종[2]이 더 증가한다. 종격동기종(±)

  • 진단은? → 기관지손상 w/ 기흉
  • 검사는? → 기관지내시경 (찢긴 부위를 찾아내어 수술 고려)

대표유형: 호흡음감소(±) ↓산소포화도(±) X-ray사진(+)

  • [3]
  • 진단은? → 혈흉 d/t 외상
  • 치료는? → 가슴관 삽입
    • if: "가슴관에서 혈액성 흉수가 계속 나온다" or 혈역학적 불안정(+)
      • 치료는? → 응급수술

대표유형: 호흡음감소(±) 위가 흉강에 있는 사진(+)

  • 진단은? → 가로막탈장
  • 치료는? → 가로막 봉합술

대표유형: 호흡음 정상(+) 사진(+)

  • 진단은? → 갈비뼈골절
  • 치료는? → 진통제(NSAID→아편유사제)

  1. 혈역학적 불안정(+) 에서는 가슴관삽입이 금기이다. 바늘 감압 먼저 하고, 이후 V/S stable해지면 가슴관삽입 시행↩︎

  2. 가슴/목 피부 아래에 공기가 차는 것. 촉진시 바스락 거린다고 제시하기도 한다.↩︎

  3. Chest PA(서서 촬영)에서는 air-fluid level이 특징적이고, Chest AP(누워서 촬영)에서는 diffuse한 모습을 보여준다.↩︎
