UTI

(전립샘염, 성병/요도염, 질염, 골반염(자궁/난소 감염) 은 각 문서 참고)

대표유형: 단백뇨(±) 혈뇨(±) 농뇨(+) 아질산염(±) 세균다수(±)

  • 원인균은? → 대장균 (m/c)[1]
  • 검체 채취법은? → 중간뇨 채취[2]
  • if: 무증상
    • 진단은? → 무증상세균뇨
    • 검사는? → 소변배양
    • 치료는?
      • if: 임신 → 암피/아목시
      • else: → 경과관찰
  • if: 배뇨통/빈뇨/절박뇨(+) 하복부 압통(±) CVAT(-) 4대 염증수치(-)
    • 진단은? → 방광염(하부요로감염)
    • 검사는? → 소변배양
    • 치료는?[3]
      • if: 임신 → 포스포/니트로
      • else: → 포스포/니트로/Cipro
  • if: CVAT(±) 4대 염증수치(+)
    • 진단은? → 신우신염(상부요로감염)
    • 검사는? → 소변배양 / CT
    • 치료는?
      • 배양 전[4]
        • if: 임신/영아 → 탁심트리악손[5]
        • else: → Cipro 트리악손[6]
          • if: 패혈성 쇽(+) → Pip-Tazo 고려
      • 배양 후
        • if: Enterococcus → 암피/아목시
        • if: 녹농균 → 항녹농균 항생제
        • else:[7] → Cipro 트리악손
    • if: 항생제 불응(+) 농양 사진(+)
      • 진단은? → 콩팥 농양(고름집)
      • 치료는? → 배농
    • if: 면역저하자(±) 기종 사진(+)
      • 진단은? → 기종성 신우신염
  • if: 도뇨관 적용 환자에서 발견된 UTI
    • 치료는? → 위 알고리즘과 정확히 똑같이 하면 됨
      • 무증상세균뇨 → 경과관찰[8]
      • 신우신염 → 같은 항생제
  • if: 선지에 "소변배양"과 "항생제"가 같이 있는 경우
    • 항생제를 투여하면 배양 양성률이 떨어지기 때문에, 투여 전에 검체를 받는다.
    • 하지만 배양 결과는 기다릴 필요 없이 바로 치료(경험적 항생제)를 시작해야 한다.
    • 배양 결과가 나오면 항생제를 조정한다.
    • 답 → 소변배양
    • 유일한 예외: 건강한 사람의 무증상세균뇨 → 배양도 안하고 치료도 안함.

대표유형: 소아 불명열(+)[9] 농뇨 (무언급/-)

  • 검사는? → 소변배양
  • 치료는? → 탁심트리악손

대표유형: 소아 농뇨(+) 비특이적 증상 이것저것(+)[10]

  • 검사는? → 소변배양 and 초음파 (구조이상 선별)
  • if: 재발성(+) or 수신증(+)
    • 검사는? → 비뇨기 조영술[11] (구조이상 확진)

대표유형: 소아 혈뇨(+) 농뇨(+) 이형적혈구>75%(-) ↑소변 칼슘/크레아티닌 비(-)

  • 진단은? → 급성출혈 UTI (MCD, PSGN, IgAN, 특발성 고칼슘뇨증 배제 가능)

  1. 증상을 성병인척 하면서 임균/클라미디아 등을 틀린선지로 제시한다↩︎

  2. 틀린 선지로 "아침첫소변" 등이 있다.↩︎

  3. 방광염이 감기처럼 흔하고 약한 질환이다보니까, 대체로 부작용이 약하고 내성을 다르게 가지는 항생제를 선택하는 경향이 큼.↩︎

  4. EC에 준한 경험적항생제↩︎

  5. 딱 한 문제 "세파졸린"이 답이었던 적이 있는데, 좋은 답은 아니었던듯 하다.↩︎

  6. "세픽심"이 답이었던적도 꽤 있다.↩︎

  7. EC를 포함해, 클렙시엘라, Enterobacter 등등↩︎

  8. 무려 틀린 선지로 "도뇨관 교체" 가 있었음. 교체할 필요 없다.↩︎

  9. 소아에서는 FUND/CVAT 확인이 어려울 수 있음. → 확인 안되면 방광염/APN을 구분하지 않고 UTI로 싸잡음. 증상은 비특이적이나 발열만 있어도 APN에 준해서 탁심트리악손 주면 됨.↩︎

  10. 울고 보챈다, 묽은변, 먹는양이 줄고 기저귀가는 횟수도 줄었다, 흉부X선사진 등등.↩︎

  11. 틀린 선지로 "콩팥요관방광 단순촬영"이 제시된다↩︎
